Medical Image Classification

Overview

The project involves classification of medical images collected from X-ray, MRI and CT scans into five categories—Hand, Breast, Head, Abdomen, and Chest—using two deep learning models: a Multi-Layer Perceptron (MLP) and a Convolutional Neural Network (CNN).

Data Collection

The dataset used was sourced from Kaggle and contains 48,954 images. The dataset comprises 10,000 images for each category except for Breast, which contains 8,954 images. Images are 64x64 pixels with single channel.

Data Preprocessing

All the images were normalized and the dataset was shuffled and split into training, validation, and test sets with 80:10:10 ratio. Let's take a look at different categories of images in the dataset.

Deep Learning Classification Modeling

All the deep learning models were built using keras in tensorflow library utilizing a sequential model.

Multi-Layer Perceptron (MLP)/Dense Neural Networks (DNN)

MLP1 model: This model consists of

  • An input layer that flattens the 64x64 images into a 4096-feature vector.
  • Two hidden layers with 25 and 15 units, and ReLU activation.
  • An output softmax classification layer with 5 units, corresponding to the 5 categories.

MLP2 model: This model consists of

  • An input layer that flattens the 64x64 images into a 4096-feature vector.
  • Three hidden layers with 50, 25 and 15 units, and ReLU activation.
  • An output softmax classification layer with 5 units, corresponding to the 5 categories.

Test Results with MLP models:

  • Both models were trained for 30 epochs, achieving a test accuracy of over 99%.
  • MLP2 model had better precision, recall and F1 scores than MLP1
  • Plots below show the comparison of the classification reports and the confusion matrix from both models. MLP1 on left and MLP2 on right

Convolutional Neural Network (CNN)

A simplified CNN model was implemented, which generally provides better results for image data due to its ability to capture spatial hierarchies in images. Different version of CNN is widely used for computer vision tasks which can build deep learning models efficiently with lesser number of shared parameters and its ability to identify different features with translational invariance.

  • On right shows the schematics of the CNN network with 4 convolutional layers followed by pooling layers.
  • Last part of the deep network is flattened and fed to a 3 layers of DNN.
  • Output of DNN connected to a softmax function
  • Valid padding and relu activation is used in all hidden layers.

Test Results with CNN:

  • The CNN model trained for 10 epochs showed an immense improvement in the classification metrics over the two MLP models, as expected with image data.
  • The misclassification of hand and chest images were reduced significantly with the CNN model. Only 3 images were misclassifed with the CNN model.

Image augmentation techniques can be used to solve this problem. However, our current CNN model is performing exeptionally well on unseen test data.

Conclusion

Both MLP models successfully classified the medical images, with the CNN outperforming exceptionally well due to its architecture being more suited for image processing. This project demonstrates a comprehensive approach to solving a multi-class image classification problem using deep learning techniques.
