Prediction of Medical Insurance Costs

Objective

The goal of this project was to build a predictive model that estimates the annual medical insurance costs for individuals based on various demographic and regional features such as age, sex, body mass index (BMI), number of children, smoker status, and region. Accurate cost predictions can help insurance companies price policies more effectively and assist individuals in understanding their potential expenses.

Data Collection

The dataset used for this project has been taken from Kaggle and consists of 1,338 records, each containing the following features:

  • Age
  • Sex
  • BMI (Body Mass Index)
  • Number of children
  • Smoker status
  • Region
  • Insurance charges (target variable)

Data Preprocessing

  • Checked for missing values, NaNs and removed duplicate rows.
  • Converted categorical variables (sex, smoker, region) into numerical format using one-hot encoding.
  • Split the dataset into training, validation and testing sets with a 60:20:20 ratio to evaluate the model's performance.
  • Standardized all the data using scikit learn.

Machine Learning Modeling

  • Implemented multiple machine learning and deep learning models to predict insurance costs based on the available features.
  • The models used include:
    1. Linear regression
    2. Polynomial regression
    3. Support vector regression
    4. Decision Trees
    5. Random Forests
    6. XGBoost
    7. Dense Neural Networks
  • Models 1-5 were generated using the Scikit Learn, XGBosst utilized xgboost library and dense neural networks used tensorflow library.
  • Used cross-validation to fine-tune the model and ensure it generalizes well to unseen data using the test dataset

Conclusion

This project successfully developed a model to predict medical insurance costs, offering valuable insights into the factors that drive expenses. From our modeling effort, DNN seems to doing the best job for predicting low insurance charges and RF for predicting high insurance charges. The model can be further improved by collecting more data or additional features. These predictions can aid insurance companies in pricing policies and help customers anticipate their healthcare expenses more accurately.
